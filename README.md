## Clone this project

```bash
> git clone https://github.com/pires12455/master.git
```

## Instale as dependências:
Antes de executar o comando abaixo, certifique-se de estar no diretório do projeto que
você acabou de clonar!!

```bash
> cd master
> bash install.sh
```

### Usage
```bash
> npm start
```
